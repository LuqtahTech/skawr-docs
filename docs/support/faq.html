❓ Frequently Asked Questions

Find answers to commonly asked questions about Skawr's platform, API, and services. Can't find what you're looking for? Contact our support team.

What is Skawr and how does it work?

Skawr is Saudi Arabia's first AI-powered marketplace aggregator that allows users to search and compare products across multiple e-commerce platforms simultaneously. Instead of visiting individual websites like Haraj, Dubizzle, or OpenSooq, users can search once and see results from all platforms.

Our platform uses advanced AI to understand Arabic and English queries, handle typos, and provide semantic search results that match user intent rather than just keywords.

Which marketplaces does Skawr support?

Currently, Skawr integrates with major Saudi Arabian marketplaces including:

  • Haraj.com.sa
  • Dubizzle Saudi Arabia
  • OpenSooq
  • Mstaml
  • And more being added regularly

We're continuously expanding our marketplace partnerships to provide comprehensive coverage of the Saudi e-commerce ecosystem.

Is Skawr free to use?

Yes, Skawr is free for consumers to use. You can search, compare prices, and access all basic features without any cost. We generate revenue through:

  • Partnership fees from marketplace integrations
  • Premium API access for businesses
  • Sponsored listings and advertising
  • White-label solutions for enterprises

How accurate are the prices and product information?

We update product information and prices in real-time using automated scraping and API integrations. Our system refreshes data every 15-30 minutes to ensure accuracy. However, we recommend verifying final pricing and availability directly with the seller before making a purchase.

If you notice outdated or incorrect information, please report it to help us improve our data quality.

How do I get an API key?

To get an API key:

  1. Sign up for a Skawr developer account
  2. Verify your email address
  3. Complete the developer application form
  4. Choose your pricing plan (Free tier available)
  5. Generate your API key from the dashboard

Free tier includes 1,000 requests per month. See our Getting Started guide for detailed instructions.

What are the API rate limits?

Rate limits depend on your plan:

  • Free Tier: 100 requests per hour, 1,000 per month
  • Starter: 1,000 requests per hour, 50,000 per month
  • Pro: 5,000 requests per hour, 500,000 per month
  • Enterprise: Custom limits based on agreement

Rate limit headers are included in all API responses to help you monitor usage.

How do I handle API errors?

Our API uses standard HTTP status codes:

  • 400: Bad Request - Check your parameters
  • 401: Unauthorized - Invalid API key
  • 429: Rate Limited - Slow down your requests
  • 500: Server Error - Try again later

All errors include detailed error messages and request IDs for debugging. Implement exponential backoff for rate limit handling.

Which programming languages are supported?

We provide official SDKs for:

  • JavaScript/Node.js: npm install skawr-sdk
  • Python: pip install skawr-python
  • PHP: composer require skawr/skawr-php
  • React/Vue: Frontend components available

You can also use our REST API directly with any language that supports HTTP requests. See our SDK documentation for examples.

Can I white-label Skawr for my business?

Yes! Our SaaS platform allows you to create white-label marketplace aggregators with:

  • Custom branding and domain
  • Configurable feature sets
  • API access and customization
  • Analytics and reporting
  • Multi-tenant architecture

Contact our sales team to discuss enterprise pricing and custom implementations.

How long does integration typically take?

Integration timeframes depend on complexity:

  • Basic API Integration: 1-2 days
  • Frontend Widget: 2-5 days
  • Full Platform Integration: 1-2 weeks
  • White-label Solution: 2-4 weeks
  • Custom Implementation: 1-3 months

We provide dedicated support during integration with technical account managers for enterprise customers.

What pricing plans are available?

Our pricing is designed to scale with your usage:

  • Free: $0/month - 1,000 requests, basic support
  • Starter: $99/month - 50,000 requests, email support
  • Pro: $499/month - 500,000 requests, priority support
  • Enterprise: Custom pricing - Unlimited requests, dedicated support

All plans include access to our search API, real-time data updates, and comprehensive documentation.

Do you offer custom pricing for high-volume usage?

Yes, we offer volume discounts and custom pricing for:

  • Businesses requiring 1M+ requests per month
  • Multi-year contracts
  • Enterprise white-label solutions
  • Strategic partnerships

Contact our sales team to discuss your specific requirements and get a custom quote.

What is the API response time?

Our API is optimized for speed:

  • Average Response Time: 200-500ms
  • 95th Percentile: < 1 second
  • Uptime SLA: 99.9%
  • Global CDN: Distributed across Middle East regions

Response times may vary based on query complexity and the number of marketplaces being searched.

Do you provide webhooks for real-time updates?

Yes, we offer webhook notifications for:

  • Price Changes: When product prices are updated
  • New Products: When matching products are discovered
  • Availability: When product availability changes
  • Custom Events: Based on your specific requirements

Webhooks are available in Pro and Enterprise plans. Configure them through the dashboard or API.

How do you handle Arabic text and RTL layouts?

Skawr is built with Arabic-first design:

  • Native Arabic Search: Full Unicode support and text normalization
  • RTL Support: Right-to-left layout handling in responses
  • Bilingual Processing: Simultaneous Arabic and English query processing
  • Cultural Context: Understanding of Saudi market terminology

Our AI models are specifically trained on Arabic e-commerce data and Saudi marketplace content.
