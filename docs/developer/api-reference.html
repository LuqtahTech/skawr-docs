🔧 API Reference

The Skawr API provides programmatic access to our marketplace aggregation platform. Use our RESTful API to search products, manage listings, and integrate with the Skawr ecosystem.

Base URL: https://api.skawr.com/v1
Authentication: Bearer Token (API Key)
Rate Limiting: 1000 requests per hour

🔑 Authentication

All API requests require authentication using an API key. Include your API key in the Authorization header:

cURL 
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.skawr.com/v1/search

🔍 Search API

GET /search

Search for products across all integrated marketplaces with AI-powered semantic search.

Query Parameters

Parameter Type Required Description
q string ✅ Search query in Arabic or English
limit integer ❌ Number of results (1-100, default: 20)
offset integer ❌ Pagination offset (default: 0)
sort string ❌ Sort by: relevance, price_asc, price_desc, date
marketplace string ❌ Filter by marketplace: haraj, dubizzle, opensooq
min_price number ❌ Minimum price filter (SAR)
max_price number ❌ Maximum price filter (SAR)

Try It Out

Example Request

cURL 
curl -G "https://api.skawr.com/v1/search" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -d "q=iPhone 15" \
     -d "limit=10" \
     -d "sort=price_asc"

Create a new tenant for white-label marketplace integration.

Request Body

JSON 
{
  "name": "AlRajhi Marketplace",
  "domain": "marketplace.alrajhi.com",

📈 Analytics API

GET /analytics/search-trends

Get search trends and popular queries for market insights.

Query Parameters

Parameter Type Required Description
period string ❌ Time period: 7d, 30d, 90d (default: 7d)
category string ❌ Filter by category
marketplace string ❌ Filter by marketplace

⚠️ Error Handling

The API uses standard HTTP status codes and returns detailed error information:

400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
429 Rate Limited - Too many requests
500 Internal Server Error

Error Response Format

JSON 
{
  "status": "error",
  "error": {
    "code": "INVALID_PARAMETER",
    "message": "The 'q' parameter is required",
    "details": {
      "parameter": "q",
      "expected": "string",
      "received": "null"
    }
  },
  "request_id": "req_xyz123"
}

📚 SDK Libraries

JavaScript/Node.js

npm install skawr-sdk
import { Skawr } from 'skawr-sdk';

const skawr = new Skawr({
  apiKey: 'your_api_key'
});

const results = await skawr.search('iPhone 15');

Python

pip install skawr-python
from skawr import Skawr

client = Skawr(api_key='your_api_key')
results = client.search('iPhone 15')

PHP

composer require skawr/skawr-php
search('iPhone 15');
