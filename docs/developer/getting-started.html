🚀 Getting Started

Quick start guide for developers integrating with Skawr platform. Get up and running with our API in minutes.

📋 Prerequisites

🔑

API Key

Obtain an API key from the Skawr admin dashboard

🔧

Development Environment

Any environment supporting HTTP requests (Python, JavaScript, cURL, etc.)

📚

REST API Knowledge

Basic understanding of REST API concepts and HTTP methods

🔐 Authentication

All Skawr API requests require authentication using an API key in the request header:

Health Check Example 
# Test your API key with a health check
curl -H "X-API-Key: your_api_key_here" \
     https://api.skawr.com/api/v1/health

🛡️ API Key Security

  • Never expose your API key in client-side code
  • Use environment variables for key storage
  • Rotate keys regularly for security
  • Monitor usage in the admin dashboard

🎯 Your First API Call

Let's make your first search request to find products across Saudi marketplaces:

1

Basic Product Search

Search for products using a simple text query:

cURL 
curl -X POST "https://api.skawr.com/api/v1/search" \
  -H "X-API-Key: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "laptop",
    "limit": 10
  }'
2

Expected Response

You'll receive a JSON response with search results:

JSON Response 
{
  "status": "success",
  "data": {
    "total": 156,
    "results": [
      {
        "id": "prod_abc123",
        "title": "Dell XPS 13 Laptop",
        "price": {
          "amount": 3299,
          "currency": "SAR"
        },
        "marketplace": {
          "name": "Haraj",
          "url": "https://haraj.com.sa/..."
        },
        "seller": {
          "name": "TechStore",
          "rating": 4.8
        },
        "location": "Riyadh"
      }
    ]
  }
}
3

Error Handling

Always handle potential API errors gracefully:

JavaScript 
const response = await fetch('https://api.skawr.com/api/v1/search', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: 'laptop',
    limit: 10
  })
});

if (!response.ok) {
  const error = await response.json();
  console.error('API Error:', error);
  return;
}

const data = await response.json();
console.log('Search Results:', data.data.results);

🎉 Quick Success Tips

🔍

Search Tips

  • Use Arabic or English queries
  • Include brand names for better results
  • Try different spelling variations
⚡

Performance

  • Use pagination for large result sets
  • Cache results when appropriate
  • Monitor your rate limits
🛠️

Development

  • Start with the sandbox environment
  • Use our official SDKs when available
  • Check status codes in responses
